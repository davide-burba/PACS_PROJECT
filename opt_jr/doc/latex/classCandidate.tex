\subsection{Detailed Description}
\hyperlink{classCandidate}{Candidate} class is an auxiliary class included in \hyperlink{classsCandidates}{s\-Candidates} which is used by local\-Search; it stores data about pairs of application and the consequent changes on the objective function after cores exchange.
